CUSTOMIZATION TIPS
==================

CHANGE COLORS
-------------
Edit css/styles.css, find :root variables:
```css
:root {
    --primary-color: #ff6b35;    /* Main orange */
    --secondary-color: #f7931e;  /* Warm orange */
    --accent-color: #ffd700;     /* Gold */
    /* ... etc ... */
}
```

UPDATE MENU ITEMS
-----------------
Edit js/data.js, update MENU_DATA array:
```javascript
{
    id: 1,
    name: "Your Item Name",
    category: "Rice",  // Must match a CATEGORIES name
    price: 2500,
    description: "Your description",
    ingredients: ["item1", "item2"],
    imageUrl: "https://...",  // Your image URL
    popularityScore: 85,       // 0-100
    featured: true             // Show on home
}
```

ADD NEW CATEGORY
----------------
1. Edit js/data.js
2. Add to CATEGORIES array:
   ```javascript
   { name: 'NewCategory', icon: 'ðŸŽ¯' }
   ```
3. Add items with category: 'NewCategory'
4. Update MENU_DATA accordingly
